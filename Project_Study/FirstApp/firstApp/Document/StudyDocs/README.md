#  사다리타기 게임

>20210611(금)

## 화면 설계
1. 인트로 
    - 시작 버튼 ( 게임생성시작)
2. 설정화면
    - 타이틀 : 게임설정 - Header
    - 화면 :  인원 설정 뷰, 당첨 설정 뷰
    ### 인원설정 뷰
    - 인원수 표시 및 입력 UI
    - 참여 인원 표시 및 입력 UI
    - 다음이동 버튼 UI
    ### 당첨 설정 뷰
    - 당첨자수 표시 및 입력 UI
    - 당첨내용 표시 및 입력 UI
    - 생성 버튼 UI
    
3. 메인화면
    - 상단 뷰 : 추가 기능 시
    - 중단 : 사다리 게임 뷰 표시
    - 하단 : 사다리 게임 시작 버튼, 사다리타기 결과 보기 버튼
